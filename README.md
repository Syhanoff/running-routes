# Running Routes

WordPress plugin to display GPX tracks on interactive maps with support for:

- Divi Builder (4 & 5)
- Gutenberg
- Classic Editor (shortcode + TinyMCE)
- ACF integration
- OpenLayers + OpenTopoMap / Mapy.cz

## Architecture

- Core: Route, Parser, Renderer
- Adapters: maps, formats, integrations
- Fully extensible
